# 5k-training-buddy

## An app that works with couch to 5k programs

My wife decided to use a couch to 5k training program to start running, but is often frustrated by having to set numerous timers on her phone for the different run/walk interval times, so I decided to build this to handle the timing for her. 

What it does:

- Allows the user to choose what session type
- Allows user to enter customized time for total run session, run intervals, walk intervals, and warmup or cooldown
- Alerts the user when to start session after a warmup, when to switch between run/walk intervals, when to start the cooldown, and when the session is over
- Breaks down the total time so user can see how much time was spent running vs walking
